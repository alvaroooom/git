Examen Git y Github

Alumn@: AMF

  1. ¿Que es un sistema de control de versiones?

    Un sistema de control de versiones es una herramienta que permite gestionar los cambios realizados en archivos a lo largo del tiempo, proporcionando la capacidad de cambiar, revertir y colaborar en proyectos de mejor manera.

  2. ¿Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones, mientras que GitHub es una plataforma basada en la nube que utiliza Git para almacenar, compartir y colaborar en proyectos.

  3. ¿Cómo creo una rama en git?

    Para crear una nueva rama en git, podemos usar el comando "git branch" y el nombre que queremos darle a la rama.

  4. ¿Cómo clono un repositorio?

    Para clonar un repositorio, podemos usar el comando git clone y la url, pudiendo usar url HTTPS o SSH.

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    Para bajar los últimos cambios de un repositorio, podemos usar la orden "git pull".

  6. ¿Cómo puedo gener un tag?

    Para generar un tag, podemos usar la orden git tag nombre_del_tag y para enviar el tag al repositorio remoto podemos usar la orden git push origin nombre_del_tag
